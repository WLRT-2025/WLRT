Author Metadata and ORCID (WLRT)

Status: active / meta-authorship

Description:
This branch contains canonical author information
associated with the WLRT framework.

Contents include:
- full author name,
- ORCID identifier,
- institutional or independent researcher status,
- preferred citation formats,
- licensing and attribution notes.

The goal is to ensure consistent and correct attribution
of WLRT-related materials across all platforms.
