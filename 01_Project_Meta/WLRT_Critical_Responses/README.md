Responses to Strong Criticism of WLRT

Status: active / meta-critical

Description:
This branch contains structured responses
to strong, informed, and critical objections to WLRT.

Topics include:
- claims of non-falsifiability,
- accusations of post-hoc explanation,
- comparison with price-based models,
- critique of LFI observability,
- philosophical objections to non-causal signals.

Each response aims to be precise, limited in scope,
and aligned with the canonical WLRT statement,
without rhetorical overreach.
