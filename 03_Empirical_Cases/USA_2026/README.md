# USA 2026

This directory contains empirical WLRT case studies analyzing the systemic resilience of the United States in 2026.

Unlike traditional macroeconomic analysis, these documents focus on redistribution capacity, compensating mechanisms, phase transitions and early signs of systemic fragility.
