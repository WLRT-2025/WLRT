# Russia 2026

This directory contains empirical WLRT case studies analyzing the systemic evolution of Russia during 2025–2026.

Unlike traditional political or macroeconomic analysis, these documents examine structural resilience, redistribution capacity, compensating mechanisms and phase transitions using Wave Liquidity Redistribution Theory (WLRT).
