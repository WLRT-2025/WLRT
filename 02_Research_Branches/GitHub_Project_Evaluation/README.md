GitHub Project Evaluation (WLRT)

Status: active / meta-evaluation

Description:
This branch evaluates the WLRT GitHub repository
from the perspective of clarity, usability, and external perception.

Topics include:
- readability of the repository structure,
- coherence of documentation and naming,
- ease of navigation for new readers,
- alignment with academic and open-source standards,
- identification of friction points for contributors or reviewers.

The goal is to improve the presentation and accessibility
of WLRT without compromising theoretical rigor.
