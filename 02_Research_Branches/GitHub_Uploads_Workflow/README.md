GitHub Uploads Workflow (WLRT)

Status: active / meta-operations

Description:
This branch defines the workflow for uploading,
updating, and maintaining WLRT-related files on GitHub.

Topics include:
- repository structure and folder conventions,
- rules for adding new branches and documents,
- commit message standards,
- versioning and file naming policies,
- handling updates, replacements, and archival.

The goal is to ensure consistency, traceability,
and long-term maintainability of the WLRT repository.
