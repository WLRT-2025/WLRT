## Overview

This document defines a minimal set of formally stated, citable results derived within the Wave Liquidity Redistribution Theory (WLRT).

These statements are designed to be:
- self-contained,
- context-independent,
- directly usable in external research.

They do not require full knowledge of the WLRT framework.

---

## 1. Non-Identifiability Principle

**Statement**

Observable price trajectories do not uniquely determine the underlying system state in liquidity-driven systems.

**Interpretation**

Multiple distinct liquidity configurations may produce identical price behavior.

---

## 2. Local Future Ambiguity

**Statement**

For a given observed price state, multiple non-equivalent short-term system evolutions may be simultaneously admissible.

**Interpretation**

The system admits a set of valid future trajectories rather than a single deterministic continuation.

---

## 3. Liquidity-State Primacy

**Statement**

Price is an observable projection of an underlying liquidity state and does not constitute a fundamental state variable of the system.

**Interpretation**

Liquidity defines the system state; price reflects it but does not encode it fully.

---

## 4. Constrained Evolution Principle

**Statement**

System evolution in liquidity-driven markets is restricted to a finite set of structurally admissible transitions.

**Interpretation**

Not all price paths are structurally valid; the system evolves within constrained configurations.

---

## 5. Fragility Amplification Condition

**Statement**

Local increases in system fragility correspond to reduced stability of admissible transitions and increased divergence of possible outcomes.

**Interpretation**

Higher fragility implies less predictable and less stable system behavior.
