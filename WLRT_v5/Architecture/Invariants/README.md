# Architectural Invariants

## Status

Research v0.1

This directory investigates possible invariants, diagnostic parameters and admissibility conditions within the architectural branch of Wave Liquidity Redistribution Theory (WLRT) v5.0.

No architectural invariant is considered established at the current research stage.

---

# Purpose

The purpose of this directory is to distinguish between:

- true invariants;
- diagnostic parameters;
- state-dependent thresholds;
- admissibility conditions;
- empirical regularities.

These concepts must not be treated as equivalent.

---

# Invariant

An invariant is a property that remains unchanged under a specified class of architectural transitions.

An invariant must always identify:

1. the measured property;
2. the transition class under which it is preserved;
3. the assumptions required for preservation;
4. the conditions under which preservation fails.

A quantity is not an invariant merely because it is useful or scale-independent.

---

# Diagnostic Parameter

A diagnostic parameter is a measurable quantity used to compare architectural states or identify changes in system condition.

A diagnostic parameter may vary during normal system operation.

The preliminary ratio between Working Configuration and Architectural Configuration belongs to this category until stronger properties are demonstrated.

---

# State-Dependent Threshold

A state-dependent threshold defines a minimum, maximum or admissible range required under specific environmental and architectural conditions.

Such a threshold may change between cycles.

The existence of a threshold does not imply the existence of a universal constant.

---

# Admissibility Condition

An admissibility condition determines whether a transition or resulting state is compatible with continued redistribution.

Examples may include conditions for:

- cycle continuation;
- value withdrawal;
- architectural mobilization;
- architectural accumulation;
- transition to the next cycle.

---

# Research Requirements

A candidate invariant or diagnostic parameter must be tested against:

- WW transitions;
- WA transitions;
- AW transitions;
- AA transitions;
- external capital inflow;
- value withdrawal;
- operational losses;
- market revaluation;
- changes in environmental conditions.

---

# Current Candidates

Current research candidates include:

- the ratio between Working and Architectural Configuration;
- minimum Working Configuration;
- minimum Architectural Configuration;
- cycle reproduction conditions;
- transition balance conditions.

These candidates are not yet canonical mathematical results.
