# Architectural States

## Status

Provisional Canonical v0.1

This directory introduces the concept of architectural states within the architectural branch of Wave Liquidity Redistribution Theory (WLRT) v5.0.

Architectural states describe the condition of an adaptive system at a specific moment in time.

A state is defined by the values of the canonical architectural variables and the relationships between them.

---

# Purpose

States provide the bridge between architectural objects and dynamic mathematical models.

They make it possible to describe how adaptive systems evolve through successive redistribution cycles.
