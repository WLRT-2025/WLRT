# Cycle Phase Model

## Status

Provisional Canonical v0.1

---

# Purpose

This document defines the canonical high-level phases of an adaptive redistribution cycle.

The model separates mandatory cycle phases from the events occurring within those phases.

---

# General Principle

A redistribution cycle consists of mandatory structural phases.

Within these phases, multiple redistribution events may occur.

Events should not be confused with phases.

---

# Phase 1

## Initialization

The initial Architectural State is established.

The following elements are identified:

- Working Configuration
- Architectural Configuration
- Total System Value
- Resource Allocation
- Environmental Conditions

This phase defines the starting state of the cycle.

---

# Phase 2

## Active Redistribution

The system performs redistribution.

Multiple events may occur.

Examples include:

WW

WA

AW

AA

External capital inflow

Losses

Operational costs

Market revaluation

No fixed sequence is assumed.

---

# Phase 3

## Reconciliation

The system determines the results of the completed redistribution activity.

Examples include:

- Created Value
- Market Revaluation
- Operational Losses
- Updated Configurations

This phase establishes the complete Architectural State before allocation decisions.

---

# Phase 4

## Architectural Allocation and Closure

The system allocates the results of the completed cycle.

Possible decisions include:

- restoring Working Configuration;
- strengthening Architectural Configuration;
- withdrawing value;
- carrying resources into the next cycle.

The next redistribution cycle begins from the resulting Architectural State.

---

# Continuous External Processes

Certain processes operate continuously throughout the cycle.

Examples include:

- market price changes;
- environmental changes;
- liquidity conditions;
- external information.

These processes are not considered independent phases.

---

# Cycle Structure

Initialization

↓

Active Redistribution

↓

Reconciliation

↓

Architectural Allocation and Closure

↓

Next Cycle

---

# Current Status

The four-phase model is considered the current canonical representation of redistribution cycles.

The internal dynamics of each phase remain an active area of research.
