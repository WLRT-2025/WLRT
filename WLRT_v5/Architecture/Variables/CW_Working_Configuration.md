# CW — Working Configuration

## Status

Provisional Canonical v0.1

---

## Symbol

CW

---

## Name

Working Configuration

---

## Preliminary Definition

Working Configuration (CW) represents the current organization of system resources that enables execution of the next redistribution cycle.

CW is determined by resource organization rather than by resource quantity.

---

## Purpose

CW supports operational redistribution.

---

## Determines

CW influences:

- immediate redistribution capability;
- operational flexibility;
- execution readiness.

---

## Does Not Describe

CW does not measure:

- total value;
- accumulated reserves;
- long-term sustainability.

---

## Current Interpretation

CW is expected to evolve after every redistribution cycle.
