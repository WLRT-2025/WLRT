# P — Created Value

## Status

Provisional Canonical v0.1

---

## Symbol

P

---

## Name

Created Value

---

## Preliminary Definition

Created Value (P) represents the additional value generated by the adaptive system during a redistribution cycle.

---

## Purpose

P measures the productive outcome of redistribution.

---

## Important Observation

The creation of value does not necessarily imply improvement of the Architectural State.

Created Value and architectural development represent different system properties.
