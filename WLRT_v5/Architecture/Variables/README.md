# Variables

## Status

Provisional Canonical v0.1

This directory defines the canonical variables used in the architectural branch of Wave Liquidity Redistribution Theory (WLRT) v5.0.

Variables are introduced only after the corresponding architectural objects have been defined.

Each variable is documented in an individual file.

The purpose of this directory is to provide precise definitions before mathematical relationships are introduced.

---

# Development Rule

Variables describe architectural objects.

They do not define them.

Objects are introduced in:

Architecture/Architectural_Objects.md

Relationships are introduced in:

Architecture/Architectural_Relationships.md

The complete system state is defined in:

Architecture/Architectural_State.md

Mathematical equations are introduced only after all canonical variables have been established.
