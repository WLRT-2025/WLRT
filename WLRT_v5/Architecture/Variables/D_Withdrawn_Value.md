# D — Withdrawn Value

## Status

Provisional Canonical v0.1

---

## Symbol

D

---

## Name

Withdrawn Value

---

## Preliminary Definition

Withdrawn Value (D) represents the value permanently removed from the adaptive system after completion of a redistribution cycle.

---

## Purpose

D describes external value extraction.

---

## Important Observation

Withdrawal of value may alter the Architectural State even when Total System Value remains relatively stable.

The relationship between D and architectural degradation is an active research topic.
